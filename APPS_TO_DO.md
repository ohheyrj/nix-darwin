# macOS Applications Management

## Overview
This document tracks macOS applications and their packaging status for Nix/NixOS configuration management.

**Legend:**
- ✅ Available and working in Nix
- 🔄 Available in Nix but needs updates/fixes
- 📦 Needs custom packaging
- 🏪 App Store only
- ❌ Cannot be packaged
